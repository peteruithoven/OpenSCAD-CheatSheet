OpenSCAD CheatSheet

Syntax

var = value; module name(…) { … }
name(); function name(…) = …
name(); include <….scad> use <….scad>

2D

circle(radius) square(size,center) square([width,height],center) polygon([points]) polygon([points],[paths])

3D

sphere(radius) cube(size) cube([width,height,depth]) cylinder(height,radius,center) cylinder(height,radius1,radius2,center) polyhedron(points, triangles, convexity)

Transformations

translate([x,y,z]) rotate([x,y,z]) scale([x,y,z]) mirror([x,y,z]) multmatrix(m) color("colorname") color([r, g, b, a]) hull() minkowski()

Boolean operations

union() difference() intersection()

Modifier Characters

*
disable
!
show only
#
highlight
%
transparent

Mathematical

abs sign acos asin atan atan2 sin cos floor round ceil ln len log lookup min max pow exp rands
